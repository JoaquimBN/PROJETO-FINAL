GIT

Git é simplesmente um sistema onde o usuário tem o controle da versão de arquivos que estão em fase de desenvolvimento onde mais de um usuário pode contribuir simultaneamente na criação de um arquivo sem que coloque em risco a versão principal.

OS PRINCIPAIS COMANDOS DE GIT

GITHUB

GitHub é a plataforma de hospedagem de arquivos de código com controle de versão por meio do Git que também permite vários usuários contribuírem simultaneamente no desenvolvimento de arquivos código onde são privados ou públicos.

O QUE É GIT E GITHUB?

DESENVOLVIMENTO ORIENTADO POR TESTES

Desenvolvimento Orientado por Testes é um meio de desenvolvimento que se baseia na aplicação de pequenos ciclos de repetição, onde tem o objetivo de desenvolver um teste para que o código seja testado e tenha um resultado positivo. Esses testes são elaborados antes mesmo dos códigos serem desenvolvidos onde os desenvolvedores fazem esses testes até falhar.

TDD (Test Driven Development)

PYTEST

Pytest é um framework onde é usado na parte de teste de modo mais dinâmico e rápido onde são desenvolvidos em Python, com isso o código fica mais compacto para que seja mais fácil para dar suporte de manutenção e seja mais legível.
